# FloristicRelayGame
## An R package to simulate the "Floristic Relay Game"
*Please* install the package using `devtools::install_github("PopGen33/FloristicRelayGame", build_vignettes = TRUE)`. This will build the 
vignette that describes how to use the package and provides examples. You can view the vignette by 
running `devtools::install_github("PopGen33/FloristicRelayGame", build_vignettes = TRUE)`.
